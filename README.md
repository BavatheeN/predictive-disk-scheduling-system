# 🚀 Predictive Disk Scheduling System

## 📌 Description
A web-based simulation system that models disk request arrivals using Poisson processes and optimizes disk scheduling using algorithms like FCFS, SCAN, and a custom Predictive Scheduler.

This project demonstrates how intelligent scheduling can reduce seek time and improve system performance.

---

## 🎯 Features
- 📊 Poisson-based disk request generation
- ⚙️ Implementation of:
  - FCFS (First Come First Serve)
  - SCAN (Elevator Algorithm)
  - Predictive Scheduling (custom logic)
- 📈 Performance analysis (seek time, efficiency)
- 🌐 Interactive web interface
- 📉 Graphical visualization of disk head movement

---

## 🧠 Concepts Used
- Operating Systems (Disk Scheduling)
- Design and Analysis of Algorithms (DAA)
- Probability (Poisson Distribution)
- Basic Machine Learning concept (Prediction logic)

---
